Box Model

Box model is a layout model for HTML elements

Which describes the content, padding, border and margin of the element

Content

The content area is where the text and images and videos appear

Padding

The padding area is the space between the content and the border

To change the padding, use the padding property in the CSS

            #box1{
            padding: 20px; /* for all sides */
            padding: 10px 20px; /* vertical horizontal */
            padding: 10px 20px 5px 15px; /* top right bottom left */
            padding-right: 20px;
            padding-left: 10px;
            padding-top: 15px;
            padding-bottom: 5px;
            }

Border

The border area is the line/layout around the content

To change the border, use the border property in the CSS

                #box1{
                border: 1px solid black;  /* shorthand property/structure  */
                border-style: solid;
                border-color: black;
                border-width: 1px;
                }

Margin

The margin area is the space outside the border

To change the margin, use the margin property in the CSS

                #box1{
                margin: 30px;
                }
