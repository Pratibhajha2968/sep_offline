List

List tags are used to have the list of items

Types are:

  1. Unordered List
  2. Ordered List
  3. Description List

Unordered List

Unordered list is used to have the list of items in unordered way

ul tag is used to have the unordered list

The Default type is disc

The type can be changed by using the type attribute

Types values are:

  • disc
  • circle
  • square

Ordered List

Ordered list is used to have the list of items in ordered way

ol tag is used to have the ordered list

The Default type is Number

The type can be changed by using the type attribute

Types values are:

  1. 1
  2. A
  3. a
  4. I
  5. i

Note : the default value need to be start from starting value

To change the starting value we can use start attribute

Description List/Data Definition List

Description list is used to have the list of items in description way

dl tag is used to have the description list

dt tag is used to have the description term

dd tag is used to have the description definition
